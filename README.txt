Additional features:

Holes as targets for the non-cue ball to hit and complete a level.

Used the Separating Axis Theorem and Collision Response physics to implement a physically accurate collision detection and response system for convex polygons and circles.

Added the two aiming lines to help line up shots.

Used gettimeofday to ensure that the animations look smooth whatever the framerate.

Added additional levels.

The game keeps track of how many shots it takes to complete a level (displayed as terminal output).

Push 'r' to enter Rainbow mode.
